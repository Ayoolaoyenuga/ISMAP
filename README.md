# ISMAP

ISMAP is a subdomain and vulnerability monitoring platform with a Flask backend and a React/Vite frontend.

## Local Development

Backend:

```powershell
cd ismap-main
.\.venv\Scripts\python.exe app.py
```

Frontend:

```powershell
cd ismap-main\frontend
npm run dev
```
